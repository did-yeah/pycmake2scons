# pycmakes2scons
Automating creation of SConscript file in a C/C++ project

to launch program:

% python -m app.main

to use mypy in pycmake2cons directory:

% mypy tests app

to test program:

% python -m pytest

to use isort in pycmake2cons directory:

% isort -rc tests app

to use black in pycmake2cons directory:

% black tests app
